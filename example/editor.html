Gettext Editor

Edit gettext .mo and .po files in your browser.

This editor works entirely in your browser, nothing is actually uploaded or downloaded. TextEncoder/TextDecoder support is proveded by stringencoding library (Firefox and FirefoxOS support this out of the box).

NB! Overwriting existing files can be a destructive action as webgettext only uses UTF-8 encoding when creating .mo and .po files. If you need your translation files to be in a different charset (eg. iso-8859-1), do not use webgettext. Anything else (comments, notes to translators, plurals etc.) should be preserved as in the source file.
